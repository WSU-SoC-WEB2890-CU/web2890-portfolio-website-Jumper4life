Pictotainment Media Inc.

Project 2 for WEB 2890

Pictotainment Media Inc. underwent a redesign to create a clean, simple website that showcases the artist’s work in a visually striking manner. The website features a dark background to enhance the vibrancy of the photos, making them stand out. Each image includes a hover feature that reveals the title when the mouse hovers over it, though this feature is disabled on mobile devices for ease of use. The site is designed for easy navigation, ensuring a seamless user experience as visitors explore the artist’s portfolio and updates.
